# 🧠 Engenharia de Software em IA Aplicada

Repositório dedicado à documentação, códigos e projetos desenvolvidos durante a pós-graduação em Engenharia de Software em IA Aplicada. 

Este repositório atua como um laboratório contínuo, unindo os fundamentos teóricos com aplicações práticas no mercado de desenvolvimento, infraestrutura e criação de produtos inteligentes.

## 🎯 Objetivo
Registrar o progresso no curso e expandir os conceitos das aulas para **projetos pessoais paralelos**. O foco aqui é construir soluções de alto desempenho, desde pipelines de dados robustos até interfaces de usuário refinadas com estética *Liquid Glass*.

## 📂 Estrutura do Repositório

### 📚 Módulos Oficiais
* **01 a 03:** Fundamentos de LLMs, Prompt Engineering Avançado e MCP (Model Context Protocol).
* **04:** Arquitetura de Agentes Autônomos (LangGraph, Memória, Reflection).
* **05 a 07:** Ferramentas de IA para UX/UI, DevOps (IaC) e Gestão de Projetos.
* **08 a 10:** Arquitetura de Sistemas (Agentic RAG), Fine-Tuning e Governança/Segurança.
* **11:** Capstone Project (Micro-SaaS).

### 🧪 Labs & Projetos Pessoais (`/labs_projetos_pessoais`)
Além dos exercícios da pós, esta pasta contém implementações proprietárias inspiradas nas aulas, como:
* Orquestração de agentes autônomos.
* Automações avançadas com Python, n8n e PostgreSQL.
* Componentes visuais modernos e painéis de controle.

## 🛠️ Stack Tecnológico Principal
* **Linguagens:** Python, JavaScript/TypeScript, SQL
* **IA & Orquestração:** OpenAI, Gemini, LangChain, LangGraph
* **Infraestrutura & Automação:** MCP, n8n, Firebase, Docker
* **Design/UI:** Padrões iOS-inspired, paleta base (Azul Profundo `#112d4e` e Laranja `#f97316`)
